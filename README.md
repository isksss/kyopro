# kyopro

競技プログラミング用リポジトリ。  

Rust習得も兼ねてRustで解く。

## AtCoder

[isksss](https://atcoder.jp/users/isksss)

## 使用
```bash
# 新規問題
$ cargo compete new <abc265>

# テスト
# 当該問題ディレクトリに移動した後に
# ex) cd atcoder/abc265
$ cargo compete test a

# 提出
$ cargo compete submit a

# 問題ページを開く
$ cargo compete open
```
[qryxip/cargo-compete](https://github.com/qryxip/cargo-compete/)
